# E-Commerce Sales Analysis (Interactive Power Bi Dashboard)
## Project Objective
The E-Commerce Sales Dashboard is an interactive Power BI report that delivers a clear, visual summary of overall business performance.
It provides insights into sales, profit, quantity, and customer segments, helping business users make data-driven decisions that improve profitability and efficiency.

## Tech Stack

Built using the following tools and technologies:

Power BI Desktop – for data visualization and dashboard creation

Power Query – for data cleaning, transformation, and preparation

DAX (Data Analysis Expressions) – for custom calculations and KPIs

Data Modeling – to connect tables and enable accurate filtering and relationships

Microsoft Bing Maps – for visualizing regional and state-level sales performance
